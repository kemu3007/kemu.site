Pyodide JavaScriptからPythonスクリプトを実行する

発行日時: 2022.05.03 02:57

最終更新日: 2022.05.05 01:04

PyCon US 2022でPyScriptについての発表があったので流れに乗り、とりあえずPyodideを手元で触ったので紹介します。

New from Anaconda: Python in the Browser

PyodideをCDN経由でインポート 

<head>
    <script src="https://cdn.jsdelivr.net/pyodide/v0.20.0/full/pyodide.js"></script>
</head>

Pythonスクリプトの実行

const pyodide = await loadPyodide(); で初期化処理を行なっており、pyodide.runPython でスクリプトの実行を行います。

runPythonの戻り値はスクリプトファイル内部で最後に評価された値です。

パッケージのインポートに関しては動作確認中

<h2>Python: <span id="python-version"></span></h2>
<hr />
<div id="python-zen"></div>

<script type="text/javascript">
async function setScriptValue(pyodide, dir, tag) {
  const script = await (await fetch(dir)).text()
  document.getElementById(tag).innerText = pyodide.runPython(script)
}
async function main(){
    const pyodide = await loadPyodide();
    setScriptValue(pyodide, "scripts/get_zen.py", "python-zen")
    setScriptValue(pyodide, "scripts/get_version.py", "python-version")
}
main();
</script>
