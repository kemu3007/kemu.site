Angular component内にscriptタグを設置する

scriptタグを src/index.html 以外に設置した場合、自動的に取り除く挙動を示します。

参考: angular/angular#4903

そのため、例えばTwitter Publishで取得できるようなボタンを設置した際、ツイートボタンのcssが当たらないなどといった不具合が発生します。

解決策

```ts import { Component, AfterViewInit } from '@angular/core';

@Component({ selector: 'app-tweet-btn', template: `tweet</a

`, }) export class TweetBtnComponent implements AfterViewInit { ngAfterViewInit(): void { const script = document.createElement('script'); script.async = true; script.src = 'https://platform.twitter.com/widgets.js'; const btn = document.getElementById('btn')!; btn.insertAdjacentElement('afterend', script); } }

```

ビューの初期化後にscriptタグを生成、指定したタグのすぐ後ろの位置にscriptタグを設置させています。
