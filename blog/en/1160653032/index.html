  1. Run GitHub Action when GitHub issue is created/updated Collecting Issues with clecle label
import requests
import json
from pathlib import Path

if __name__ == "__main__":
    Path("articles/list.json").unlink(missing_ok=True)
    res = requests.get("https://api.github.com/repos/kemu3007/portal/issues?labels=article")
    articles = json.loads(res.content)
    with open("articles/list.json", "x") as f:
        f.write(json.dumps(articles))
  1. Save the collected issue in json format and setngular to reference the article

I used Octokit to use the GitHub Rest API, but I decided to use this format because it is difficult to restrict the rate when playing at hand.

angular- cli- ghpages Automatically transitions to 404 - > target path when redirecting is done except /. Please do not forget to use the arguments of - base- href and - - cname.

  • basehrefドメイン(- - base- href={repo_name})
  • cnameドメイン(- - cname=portal.kemu.site)

You don’t have to worry about it because it is automatically issued a certificate of https. (If you use a custom domain, you need to create a CNAME record and turn it to {user_id}/github/io. )
