Pyodide Run Python scripts from JavaScript

発行日時: 2022.05.03 02:57

最終更新日: 2022.05.05 01:04

I went with the flow after the presentation on PyScript at PyCon US 2022 and introduced Pyodide, which I have touched at hand for now.

[New from Anaconda: Python in the Browser](https://www.anaconda.com/blog/pyscript- python- in- the- browser)

Import Pyodide via CDN 

<head>
    <script src="https://cdn.jsdelivr.net/pyodide/v0.20.0/full/pyodide.js"></script>
</head>

Running Python scripts

Initialization is performed with const pyodide = await loadPyodide(); and the script is executed with pyodide.runPython.

The return value of runPython is the last value evaluated inside the script file.

Importing packages is still being tested.

<script type="text/javascript">
async function setScriptValue(pyodide, dir, tag) {
  const script = await (await fetch(dir)).text()
  document.getElementById(tag).innerText = pyodide.runPython(script)
}
async function main(){
    const pyodide = await loadPyodide();
    setScriptValue(pyodide, "scripts/get_zen.py", "python-zen")
    setScriptValue(pyodide, "scripts/get_version.py", "python-version")
}
main();
</script>
