backports.zoneinfo

UniqueConstraint Expression

It is now possible to specify desc() and asc() when setting UniqueConstraint to stretch the index.

class TestModel(models.Model):
    code_1 = models.CharField(max_length=32)
    code_2 = models.CharField(max_length=32)

    class Meta:
        constraints = [
            UniqueConstraint(
                Lower('code_1'),
                Lower('code_2').desc(),
                name='code_1_2_unique',
            ),
        ]

SQL to be issued

CREATE UNIQUE INDEX "code_1_2_unique" ON "common_testmodel" ((LOWER("code_1")), (LOWER("code_2")) DESC);

QuerySet.contains

You can check whether the corresponding model exists in the QuerySet.

TestModel.objects.contains(TestModel(id=1))

SELECT (1) AS "a"
  FROM "common_testmodel"
 WHERE "common_testmodel"."id" = 1
 LIMIT 1

The query executed seems to be no different than exists(). (if this is the extent of the processing)
