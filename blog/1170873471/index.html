### ライブラリ - [@ngneat/reactive-forms](https://github.com/ngneat/reactive-forms) - 型安全にformGroupを記載 - フォームの入力内容をsession storageに保存しリロードされたタイミング等で保存/ロード - [@datorama/akita](https://github.com/datorama/akita) - 状態管理ライブラリ、リクエストの結果を非同期で処理する際にsubscribeのネストが深くなりにくいためコードが読みやすくなる - ngOnInitでhttpRequest().subscribe()と記載し、ストアから非同期的に値を取得する処理を楽にかける - 現在いるページ情報などリロード時に保持してほしい情報をストアに保存し、ロード処理の記載を楽に記載可能 - ※今サイトを見ると[@ngneat/elf](https://github.com/ngneat/elf)について記載されていたので今度試す - [ng2-adsense](https://github.com/scttcper/ng2-adsense) - scriptタグをAngularがエスケープするため自前実装するのであればdom操作する必要があるところをやってくれる - [ng-recaptcha](https://github.com/DethAriel/ng-recaptcha) - botには見てほしくない情報、アクセスを防ぎたい場合に使えるreCAPTCHAを設置可能（サイトキーについてはGCPで作成する必要あり） - [@ng-bootstrap/ng-bootstrap](https://github.com/ng-bootstrap/ng-bootstrap) - Angular向けbootstrapライブラリ - [@compodoc/compodoc](https://github.com/compodoc/compodoc) - Angularアプリケーションのドキュメントを自動生成してくれる。 - 利用する際は以下のような設定ファイルを追加し、`npx @compodoc/compodoc -p ./tsconfig.doc.json -d src/doc` - 作成されるドキュメント -> [サンプル](https://portal.kemu.site/doc/) `tsconfig.doc.json` ```json { "include": ["src/**/*.ts"], "exclude": ["src/test.ts", "src/**/*.spec.ts"] } ``` --- ### コード - メッセージを表示 -> 5秒後に削除 ```ts import { Injectable } from '@angular/core';
