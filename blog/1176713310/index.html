IP Address Checkerをリリースしました

発行日時: 2022.03.22 12:29

最終更新日: 2022.03.24 12:44

閲覧ユーザのIPアドレス等を確認できる機能をリリースしました。

IPアドレスの収集は行なっておらず、ローカルストレージに前回の記録を残していくだけのため安心してご利用ください。

内部的な構成は以下の通りです。

以下から利用することが可能です。

IP Address Checker

今回もまたRoute53のエイリアスAレコードを利用したのですがやはりエイリアスで指定すると2つのAレコードが自動で作成されるようになっているようです。

エイリアスAレコードの詳細についてはこちらで触れているので参照してください。

$ dig api.kemu.site

; <<>> DiG 9.10.6 <<>> api.kemu.site
;; global options: +cmd
;; Got answer:
;; ->>HEADER<<- opcode: QUERY, status: NOERROR, id: 56004
;; flags: qr rd ra; QUERY: 1, ANSWER: 2, AUTHORITY: 0, ADDITIONAL: 1

;; OPT PSEUDOSECTION:
; EDNS: version: 0, flags:; udp: 512
;; QUESTION SECTION:
;api.kemu.site.                 IN      A

;; ANSWER SECTION:
api.kemu.site.          60      IN      A       54.168.114.234
api.kemu.site.          60      IN      A       13.112.219.250

;; Query time: 100 msec
;; SERVER: 2404:1a8:7f01:b::3#53(2404:1a8:7f01:b::3)
;; WHEN: Tue Mar 22 21:58:08 JST 2022
;; MSG SIZE  rcvd: 74

※ API Gatewayを利用した際はIP直リンは弾かれるようになっているらしい、WAFをわざわざ用意しなくて良いので便利
