MacOS Windowsインストールメディアの作成

Windows10のダウンロード

Windows10 ISO

boot media の作成

上記サイトでダウンロードしたISOをダブルクリックしてマウントしてください。その後以下の通りにスクリプトを実行します。

GPT -> MBRにするとうまくいくこともあります。

bash $ # メディアのフォーマット $ diskutil eraseDisk MS-DOS "WIN10" GPT /dev/disk4 $ # sources/install.wim以外のファイルを転送 $ rsync -avh --progress --exclude=sources/install.wim /Volumes/CCCOMA_X64FRE_JA-JP_DV9/ /Volumes/WIN10 $ # sources/install.wimを分割して書き込み $ wimlib-imagex split /Volumes/CCCOMA_X64FRE_JA-JP_DV9/sources/install.wim /Volumes/WIN10/sources/install.swm 3200 $ # /dev/disk4のマウント解除 $ diskutil unmountDisk /dev/disk4

※ /dev/disk4 は該当する外部USBメディアのディレクトリに変更してください。($ diskutil list で現在接続されているディスク情報を確認できます。) ※ wimlibをインストールしていない場合は $ brew install wimlib を実行してインストールしてください。

実行後は通常のWindowsインストール手順と同一です。

所感

ddコマンドで簡単に作れていた記憶があったのにいつの間にこんなに複雑になってしまったのか；；

install.wim が4GBを超えてFAT32の上限(4GB)を超えてしまったからこうなっているらしい。

swm拡張子はwimファイルを分割する際に使われるらしい、初めて知った。
