Python离线翻译 argostranslate

発行日時: 2022.06.02 03:27

最終更新日: 2022.06.02 03:27

当我们在将我们的博客翻译成许多语言时，我们使用argostranslate进行离线翻译，并分享我们是如何做到这一点。

＃＃介绍 

$ brew install cmake # M1にインストールする際は必要なので他の環境では未確認
$ pip install argostranslate

##更新和下载所学模型

import argostranslate.package
import tqdm

argostranslate.package.update_package_index()
available_packages = argostranslate.package.get_available_packages()

for package in tqdm(list(filter(lambda x: x.from_code == "ja" and x.to_code == "en", available_packages))):
    argostranslate.package.install_from_path(package.download())

tqdm仅用于可视化下载状态，不需要使用。

这一次，日语- >英语被测试。

import argostranslate.package
import argostranslate.translate
from argostranslate.translate import Language, CachedTranslation


installed_languages: List[Language] = argostranslate.translate.get_installed_languages()

from_lang: Language = list(filter(lambda x: x.code == "ja", installed_languages))[0]
to_lang: Language = list(filter(lambda x: x.code == "en", installed_languages))[0]

translator: CachedTranslation = from_lang.get_translation(to_lang)

translator.translate("こんにちわ")
// -> 'Home'

translator.translate("私は日本人です。")
// -> 'Japanese'

translator.translate("ブログの多言語対応を行なっている際、argostranslateを利用したオフライン翻訳をおこなったためやり方を共有します。")
// -> 'We share how to use argostranslate for offline translation.'

translator.translate("中国語に変換したい場合はJapanese -> English -> Chineseに変換する必要があります。")
// -> 'You need to convert to Japanese -> English -> Chinese.'

translator.translate("ちょっと精度が微妙だ...")
// -> 'A little precision is subtle...'

虽然句子的省略很明显，但在正常化的文本中，这并不是一个坏印象。

然而，有许多地方被扣住了，这给人的印象是，从传达正确含义的角度来看，它不是很好。

例子)

translator.translate("IOS Safari 15.4がリリースされたのでCSS新要素を試す(svh, lvh, dvh)")
// -> 'IOS Safari 15.4 has been released.'

因此，如果你是出于避免API使用限制的需要，你可能要考虑使用argostranslate，它可以离线使用。
