RSSの作成 / Chrome Web Feederの動作確認

RSSの作成

XMLが作成できればなんでもいいので今回は専用のライブラリなどといったgeneratorを使わずにCIでスクリプトを回し作成

  • 構造(サンプル)

xml <rss version="2.0"> <channel> <title>Kemu Tech Blog</title> <link>https://portal.kemu.site/blog/</link> <description>Kemu Tech Blogでは主にDjango / Angularに関する技術的な知見のメモ、共有を行っています。</description> <language>ja</language> <copyright>...</copyright> <managingEditor>...</managingEditor> <webMaster>...</webMaster> <item> <title>Django JWT認証を利用する</title> <link>https://portal.kemu.site/blog/1210913643/</link> <description>Django Rest Framework, SimpleJWTのライブラリ動作確認を行ったのでそのメモ書き。トークンの作成 -> トークンの有効性検証 -> トークンの再発行処理 -> トークンを用いたログイン必須ページへのアクセス方法の確認を行っています。</description> <pubDate>Thu, 21 Apr 2022 11:40:05 GMT</pubDate> </item> <item> ... </item> ... </channel> </rss>

  • RSS作成スクリプト

```python from datetime import datetime from pathlib import Path from typing import Any, Dict from xml.etree import ElementTree

def append_child(parent: ElementTree.Element, tag: str, text: str): child = ElementTree.SubElement(parent, tag) child.text = text

if name == "main": rss = ElementTree.Element("rss") rss.set("version", "2.0") channel = ElementTree.SubElement(rss, "channel")

append_child(channel, "title", {{ site name }})
append_child(channel, "link", {{ site url }})
append_child(channel, "description", {{ description }})
append_child(channel, "language", "ja")
append_child(channel, "copyright", {{ copyright }})
append_child(channel, "managingEditor", {{ email }})
append_child(channel, "webMaster", {{ email }})

for article in articles:
    item = ElementTree.SubElement(channel, "item")

    append_child(item, "title", article.title)
    append_child(item, "link", article.url)
    append_child(item, article.description)
    append_child(item, "pubDate", article.created_at.strftime("%a, %d %b %Y %H:%M:%S GMT"))

ElementTree.ElementTree(element=rss).write("portal/src/rss.xml", encoding="utf-8", xml_declaration=True)

```

RSSを作成したらブラウザに存在を伝える必要があるため以下のタグを index.html に追加

html <link rel="alternate" type="application/rss+xml" title="Kemu Tech Blog RSS" href="https://portal.kemu.site/rss.xml" />

RSSをフォローし、サイトの変更を検知させる

Androidのスマートフォンから chrome://flags#web-feed にアクセスし、Web FeedフラグをEnabledになるように切り替えます。

その後、RSSを用意したサイトを確認し、右上の3点リーダをクリックし、以下の画面のように +フォロー という表記があればOKです。
