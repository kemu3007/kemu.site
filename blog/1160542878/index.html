MarkdownをHTMLに変換/cssを付与する

発行日時: 2022.03.06 07:44

最終更新日: 2022.03.06 03:21

markdownは以下のライブラリを使うことでHTMLに変換可能です。
Python-Markdown

install 

> pip install markdown

HTMLへの変換 

>>> markdown.markdown("## test")
'<h2>test</h2>'

このように変換することができます。ただ、コードブロックを伴うmarkdownの場合py-gfm, pygmentsを利用すると自動的にクラスが付与されます。(通称Github Flavored Markdown)

markdown

```python
def test(sample: str):
return sample[0]
```

出力されるhtml

  • extentions指定なし
    <p><code>python\ndef test(sample: str):\n    return sample[0]</code></p>
  • py-gfm, pygmentsをインストールし、extensionsにmdx_gfm.GithubFlavoredMarkdownExtensionを指定
    • pygmentsをインストールしない場合、出力が異なるため注意
<div class="highlight"><pre><span></span><code><span class="k">def</span> <span class="nf">test</span><span class="p">(</span><span class="n">sample</span><span class="p">:</span> <span class="nb">str</span><span class="p">):</span>\n    <span class="k">return</span> <span class="n">sample</span><span class="p">[</span><span class="mi">0</span><span class="p">]</span>\n</code></pre></div>

cssの生成 

> pygmentize -S default -f html -a .highlight > default.css
