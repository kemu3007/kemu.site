MarkdownをHTMLに変換/cssを付与する

markdownは以下のライブラリを使うことでHTMLに変換可能です。 [Python-Markdown](https://github.com/Python-Markdown/markdown/) ### install ```bash > pip install markdown ``` ### HTMLへの変換 ```bash >>> markdown.markdown("## test") '

test

' ``` このように変換することができます。ただ、コードブロックを伴うmarkdownの場合`py-gfm`, `pygments`を利用すると自動的にクラスが付与されます。(通称Github Flavored Markdown) ## markdown \`\`\`python def test(sample: str): return sample[0] \`\`\` ## 出力されるhtml - extentions指定なし ```html

python\ndef test(sample: str):\n return sample[0]

``` - `py-gfm`, `pygments`をインストールし、`extensions`に`mdx_gfm.GithubFlavoredMarkdownExtension`を指定 - `pygments`をインストールしない場合、出力が異なるため注意 ```html 
def test(sample: str):\n    return sample[0]\n
``` ## cssの生成 ```bash > pygmentize -S default -f html -a .highlight > default.css ```
