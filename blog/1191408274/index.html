テキストエリアタグ内部でのTabキーの動作を管理する

Textarea内でのTabキーの動作を変更する方法のメモです。

通常、Textarea内部でTabキーを利用した場合次の入力要素にフォーカスが動く挙動をします。そのためkeydownイベントを受け取り、指定した位置に\tを挿入するようにします。

実際のコード

app.component.html

```html

```

app.component.ts

```ts export class AppComponent { form = new FormGroup({ markdown: new FormControl('') })

handleKeyDown(event: KeyboardEvent, mdInput: HTMLTextAreaElement) {
    if (event.key === 'Tab') {
        /** Tabキー押下時のフォーカス遷移をなくす */
        event.preventDefault();
        /** HTMLTextAreaElementから現在のカーソル位置を取得 */
        const defaultLocation = mdInput.selectionStart;
        const left = this.form.controls.markdown.value.slice(0, mdInput.selectionStart);
        const right = this.form.controls.markdown.value.slice(mdInput.selectionStart);
        this.form.controls.markdown.setValue(left + '\t' + right);
        /** テキストエリア内のカーソルをtabの後に移動 */
        mdInput.selectionEnd = defaultLocation + 1;
    }
}

} ```

app.component.scss

\tの表示幅に関しては以下のcssで管理可能です。

```scss

mdInput {

tab-size: 2;

} ```
