Pillow 日本語フォントを利用する

2022/3/20にモリサワのBIZ UDフォントがSILライセンスで提供され利用可能となったので利用する方法のメモ。

モリサワ 「Google Fonts」へユニバーサルデザインに配慮した「BIZ UDフォント」を提供

```python from PIL import ImageFont from urllib.request import urlopen

font = ImageFont.truetype( urlopen("https://github.com/googlefonts/morisawa-biz-ud-mincho/raw/main/fonts/ttf/BIZUDPMincho-Regular.ttf") ) ```

TrueTypeFontの読み取り通常はホストOSの.ttfファイルを参照するのですが、bytes型を指定することが可能なためurlopenを利用し、github上のファイルを参照するようにしました。

これによりホストOSのフォント状況に依存することがなくなりストレスフリーに画像生成を行うことができるようになります。
