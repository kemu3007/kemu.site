IOS Safari 15.4 リリースされたのでCSS新要素を試す svh, lvh, dvh

発行日時: 2022.05.29 02:40

最終更新日: 2022.06.02 05:34

iPhoneでコンテンツの高さを vh に基づいて決定している場合、これまではURLバーの表示/非表示を感知できず、不要なスクロールが発生してしまうという問題がこれまでありました。

これまでの対処法は以下の形で対応してきました。

.container {
    height: -webkit-fill-available
}

このやり方の問題点として calc() の引数にできず、計算処理で高さを決定する場合、とてもつらいケースが存在していました。

今回、15.4のリリースにより解決されたのでやり方を紹介します。

Safari 15.4 Release Note

この中でもsvh, lvh, dvhについて記載します。

  • Small Viewport(svh)
    • ディスプレイサイズからURLバーなどの高さを全て引いた値
  • Large Viewport(lvh)
    • ディスプレイサイズから最小化されたURLバーなどの高さを全て引いた値
  • Dynamic Viewport(dvh)
    • ディスプレイサイズからURLバーのサイズごとに変動する値
