SAML Request / SAML Response を読む

発行日時: 2023.07.20 05:53

最終更新日: 2023.07.23 05:00

SAML Request 及び SAML Response は通常、Base64 エンコーディングが行われています。
詳細な読み方についてメモ書きを残しておきますmm

読解元となる Base64 文字列については開発者コンソール等より入手ください。[1]
今回の検証には SP に Amazon Cognito を、SAML Idp に IAM Identity Center を利用しています。

SAML Request

デコード方法例)

import base64, zlib

raw_saml_request = input()

print(zlib.decompress(base64.b64decode(raw_saml_request.encode()), -zlib.MAX_WBITS).decode())

デコード方法例)

import base64

raw_saml_response = input()

print(base64.b64decode(raw_saml_response.encode()).decode())
