Nuxtでキャッシュとうまくやる

自動的に現在のコードが最新かどうかチェックし、キャッシュの更新を行います。

検証環境

  • Nuxt: 2.15.8
  • @nuxtjs/pwa: 3.3.5

コード

nuxt.config.js

js { plugins: ['plugins/prompt-update.js'] }

plugins/prompt-update.js

js export default async () => { if ('serviceWorker' in navigator) { navigator.serviceWorker.getRegistration("/sw.js").then(registration => { setInterval(()=>{registration.update()}, 1000) registration.onupdatefound = () => window.location.reload() }) } }

setInterval の 1000 の部分は更新のチェックを行うスパンを任意に指定してください(ミリ秒) registration.update() により、/sw.jsへリクエストが送られ、内部のrevisionの値が変更されたことを検知し、リロードする処理を行っています。

動作確認手法

npm run generate, http-server dist/ -p 3000で配信を行い、内部のファイルを変更後再度generateすることで自動的に更新されることが確認できます。
