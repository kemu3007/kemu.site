Google AdsenseをAngularで利用する

通常のSSRではGoogle Adsense用の自動広告スクリプトを `` 内に記載するだけで広告を配信し、収益化させることが可能です。 ただし、SPAの場合自動広告のみを利用するとページ遷移によるリロードが発生しないため、ページ下部の固定広告のみになってしまうなど表示される広告数が多くの場合減少してしまいます。 そこで、[ng2-adsense](https://www.npmjs.com/package/ng2-adsense) を利用することで広告枠を手動で設定することができるようになります。 ### ng2-adsense導入 環境: Angular v13 1. `npm install ng2-adsense` 2. `app.module.ts`に設定を記載(広告を表示させたいコンポーネントに紐づくmodule、もしくは共通して利用する用に作成したmoduleで行ってください。) ```ts @NgModule({ declarations: [AppComponent], imports: [ BrowserModule, AppRoutingModule, AdsenseModule.forRoot({ adClient: 'ca-pub-**************' }), ], providers: [], bootstrap: [AppComponent], }) export class AppModule {} ``` 4. 表示したい箇所に表示したいレイアウトのコンポーネントを追加 ```html ``` slot, layoutKeyなどの設定はGoogle AdSenseから設定したものを利用してください。 ![image](https://user-images.githubusercontent.com/29157528/158059598-f309054a-618f-46aa-aa80-b013a392894b.png) ![image](https://user-images.githubusercontent.com/29157528/158059662-bfd1a4d8-778c-4e6c-8b63-83db0e7c8402.png) ^で表示されているslotなどが対応しています。 上記の設定を行うことで広告を表示することができます。 SPAでも自動広告が利用できるようになってほしいですがページ遷移に挟み込むタイプの広告のことなども考えると少し厳しそうな気がします。
