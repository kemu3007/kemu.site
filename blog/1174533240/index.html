Angular reCAPTCHAを利用する

reCAPTCHAを用いてbotからのアクセスを防ぐことができるようになります。

通常、reCAPTCHAを利用する場合、scriptタグをHTMLに仕込む必要があります。ただし、index.html上のscriptタグはエスケープされ、消されてしまうためng-recaptchaを利用します。

install

zsh $ npm i ng-recaptcha

introduction

  • *.module.ts

```ts import { RecaptchaModule } from 'ng-recaptcha';

@NgModule({ declarations: [...], imports: [..., RecaptchaModule], }) ```

  • *.component.html
  • siteKeyにはGCP reCAPTCHAで作成したドメインを入力してください。

```html <re-captcha (resolved)="()" siteKey="-***"

```

※テスト用キーとデプロイ用キーを設定することを推奨します。ローカルで動かす際に動作確認するため、reCAPTCHAドメインに localhost を追加する必要がありますが追加してしまうと第三者がキーを利用してローカルで動かすなどが可能になり意図しない課金が発生してしまうことが起こりうるためです。

指定したドメイン外で利用した場合以下のような表記になります。

料金については~1000000回/月まで無料のためほとんどのユースケースでは無料枠で収まるかと思います。
