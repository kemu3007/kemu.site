Tools

STAR Note
面接の際のネタ帳です。Situation, Task, Action, Resultに分類して複数記載することが可能です。
FX Caluculator
FXに利用する証拠金率、ロスカット値、購入可能の最大ロット数などの値を計算します。
Mermaid Live Editor
Mermaidをリアルタイムで描画し、記述を支援します。
OpenApi Viewer
OpenApi JSON/YAMLをSwagger-UI / Redoc形式で表示します。
Year Checker
誕生日をもとに各種卒業年度/厄年といった年月日を計算します。
Regex Checker
正規表現とテキストを入力することでテキストを対象とした正規表現のテストを行い、index, マッチ文字列を返却します。
GitHub Flavored Markdown Writer
GitHub Flavored Markdown形式のMarkdownを記載することができ、HTMLに変換可能です。
Base64 Encoder/Decoder
Base64フォーマットのデータをエンコード/デコードします。
JSON Formatter
未整形のJSONデータを整形します。
REALTIME
現在時刻を100ms単位で更新し、正確な日時をお知らせします。
JSON Typer
JSONの値をTypeScriptのinterface型に変換します。
Amazon Associate Linker
Amazonリンクにアソシエイト情報を付与します。
IP Address Checker
IPアドレス等のデータを表示します。過去の履歴も表示可能です。
